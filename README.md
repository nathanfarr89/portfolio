# Nate Farr — Portfolio

Personal portfolio site built with React, TypeScript, Vite, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Adding projects

Edit [`src/data/projects.ts`](src/data/projects.ts) — the Projects section
switches from its "coming soon" placeholder to a project grid automatically
once entries are added.

## Deployment

This is a standard Vite app — deploys to Vercel with zero configuration
(framework auto-detected, build command `npm run build`, output `dist`).
